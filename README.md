# 언스타그램
* 인스타그램 기능 스프링 부트로 구현해보기

# 개발 환경
* Spring Boot
* MacOS
* InteliJ IDEA

# 방향
* Test Driven Develop 방식을 경험해보자.

* 글 게시
* 작성시간 포함
